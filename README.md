# StudyTodo

StudyTodo is a simple command-line todo list program written in C++.

This is my first C++ project. I built it to practice basic programming concepts such as structs, vectors, loops, functions, and user input.

## Features

- Add tasks
- List all tasks
- Mark tasks as done
- Show task status with `[ ]` and `[x]`
- Exit the program from the menu

Study Todo
1. Add Task
2. List tasks
3. Mark task as done
4. Exit
